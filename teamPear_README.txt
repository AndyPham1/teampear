Files have to be compiled using Qmake or within the Qt-IDE.
Will not work with Make or Maven!

RUN REQUIREMENTS:
In order for the executable to function properly, you MUST have the publication .csv 
file within the same file location as the executable itself.
The .csv file MUST also be of a publication type AND be named "Publications_sample.csv"

NOTE:
The summary tree is not fully functional. Only one professor is displayed.
The file name and path are hardcoded into the code at the moment.
There are no labels for the pie char visualization.
